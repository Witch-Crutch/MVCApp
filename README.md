# WitchCrutch
 MVC application based on java Servlets technology. Semester work torelochka and realsanya.
## Stack
- ### Frontend 
  - HTML / JS / CSS
  - Bootstrap v4.5.2
  - JQuery v3.5.1
  
- ### Backend
  - Java Servlets v.4.0.1 
  - Freemarker template engine v2.3.30
  - JDBS PostgreSQL v.42.2.16
